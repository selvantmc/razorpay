# Flutter App

A new Flutter application.

## Getting Started

This project is a Flutter application with Material 3 design.

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- Dart SDK

### Running the app

```bash
flutter pub get
flutter run
```

### Building the app

```bash
flutter build apk        # For Android
flutter build ios        # For iOS
flutter build web        # For Web
```
